Independent Scrolling Columns.

Independent scrolling columns are essential for any major content serving website. For eg, youtube and gmail have independent scorlling sections

But if you look closely, you will notice that not all portions of the page are freely scrollable for eg, the top bar in youtube remains static throughout the app

There is also this thing about responsiveness. youtube is technically responsive as it hides the

Challanges

Challange 1

  1. the page consists 3 columns (20%,50% and 30% width) and nothing else,each column indepenedently scrollable
  2. column content:
    1. 1 small box of title text and button
    2. box should be small enough to fit 2-3 boxes in one row(therefore, flexbox)
  3. Responsiveness: On changing to mobile width:
    1. column 3 should hide, col 1 and 2 should take full width
    2. the column 1 should convert to scrollable row ,
    3. (Hard): col 1 should come below col 2

Challenge Result

Solution:

1. about parent and childrent

  1. Parent 'freeflow-parent' and children('w-20','w-50' and 'w-30') are all plain divs
  2. the w-x columns have limited width
  3. We want each of these limited width columns in the same row, so we apply float:left. this way they take wahtever space id left of the previous column
  4. (why not use flexbox?)
  5. the height part is important here since height needs to be a constant number somehow to get the internal scroll to work.
      IMPORTANT: height can't be 100% or auto or any other % since it will end up taking the complete space for content and therefore not showing the scroll
  6. For responsivenes we use mobile media queries to convert the flex direction to column, and decreseas the height of first column while keeping width to 100%
