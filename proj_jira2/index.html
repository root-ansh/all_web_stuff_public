  • Todos is a simple note taking app with priority management.
  • Press new note to create a note
  • You can add a title , details and a priority to each note.
  • in sequential mode, the note's priority will be visible, you can simple change its priority from there.
  • in sequential mode, there is also an option to filter by presing the icon
  • in slotted mode, the note's will be sorted based on priority in different sections.
  • you can edit a note's content by pressing the icon on each note
  • you can delete one or multiple notes by pressing icon on the note and then deleting the note from the top
